# 📊 Simulador de Portafolio de Inversión

Herramienta interactiva en Excel para construir y analizar 
portafolios de inversión según perfil de riesgo.

## ¿Qué hace?
- Permite distribuir capital entre 5 activos reales: CSPX, EIMI, TES, BTC y Oro
- Calcula rentabilidad esperada y volatilidad ponderada del portafolio
- Clasifica automáticamente el perfil: Conservador, Moderado o Agresivo
- Simula el crecimiento histórico de $1.000 USD entre 2015 y 2025

## Activos incluidos
| Activo | Tipo | Retorno anual histórico |
|--------|------|------------------------|
| CSPX | Renta variable — S&P 500 | 12% |
| EIMI | Renta variable — Emergentes | 6% |
| TES | Renta fija — Colombia | 8% |
| BTC | Criptoactivo | 40% |
| Oro | Commodity | 7% |

## Tecnologías
- Google Sheets / Excel
- Fórmulas financieras: SUMPRODUCT, SI anidados
- Visualización con gráficas de líneas
